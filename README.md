## Getting Started

### Prerequisites

- Python 3.8 or higher
- Git
- Required dependencies (see Installation section)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Nirman-Khairnar/Ocean-Transworld-40-Step-Automation.git
   cd Ocean-Transworld-40-Step-Automation
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To run the automation workflow:

```bash
python main.py
```
